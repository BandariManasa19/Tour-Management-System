About the Project

This project is designed as a complete student capstone. It combines a Flask backend, Bootstrap frontend, MySQL database, Random Forest-based prediction, and GenAI-powered itinerary generation. The goal is to make a realistic tour booking platform that is easy to run and easy to explain.

The app covers everything from user registration and package browsing to admin management and analytics. The AI modules add value by recommending travel categories and generating personalized itinerary ideas dynamically.

Core Features

  • User and admin authentication
  • Package management and booking workflow
  • ML-based booking probability prediction
  • AI itinerary generator
  • AI chatbot for travel help
  • Analytics and reporting dashboard
